# 🫖 Tea Shop E-commerce Website

An e-commerce tea shop built with **Next.js**, **Tailwind CSS**, and a **headless WordPress** backend.

---

## 🛠 Tech Stack

### Frontend
- Next.js  
- Tailwind CSS  

### Backend
- WordPress (Headless CMS)

---

## ✅ Current Status

The following pages are **completed**:
- Homepage
- Shop page
- Product details page
- Cart page
- About page
- Contact page
- Account page
- Checkout page
